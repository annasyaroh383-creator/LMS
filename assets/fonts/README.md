# Adding Poppins Font

To use the Poppins font in this application, please download the Poppins font files and place them in this directory:

1. Download Poppins-Regular.ttf and Poppins-Bold.ttf from [Google Fonts](https://fonts.google.com/specimen/Poppins)
2. Place the files in this directory:
   - `assets/fonts/Poppins-Regular.ttf`
   - `assets/fonts/Poppins-Bold.ttf`

After adding the font files, run `flutter pub get` to update the dependencies.
